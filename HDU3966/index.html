# 1 题目大意及思路

一颗 N 个结点的树，给你 P 次操作

操作 I ：结点 C1–>C2 之间的所有结点权值加 K；

操作 D：结点 C1->C2 之间的所有结点权值减 K；

操作 Q：询问结点 C 的值。

思路：这一眼看上去就知道是一道裸的树链剖分 (点操作)，就不多说了直接上代码。没学过树链剖分的建议先入门再来看代码。
